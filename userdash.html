UserDetails Component Documentation

    The UserDetails component in this code snippet is a React functional component designed to display and manage detailed information about a user in an admin dashboard. It allows viewing user details, editing user information, connecting with a team, and deleting user accounts.

    Dependencies:

    The component imports various UI components and utilities:
  • Custom UI components like AdminConnectWithTeam, Button, Input, Textarea, Table, TableHeader, TableBody, TableCell, and TableRow from @/components/ui.
  • React hooks such as useEffect, useState, useRef for managing component state and side effects.
  • useToast for displaying toast notifications.
  • useCookies from @/hooks/useCookies for handling cookies.
  • notFound and useRouter from next/navigation for navigation and error handling.

    • Props

  • params: An object containing type (string) and userId (string) as route parameters to fetch and display user details.

    • Component Structure

    The UserDetails component renders:
  • User Details Table: Displays basic information about the user, including ID, name, email, phone number, and status (active or inactive).
  • Editable Form: Allows editing of user email, phone number, and address through form inputs (Input and Textarea components).
    Toggles between read-only and editable mode using the readOnly state.
    Submits changes via a form submission handler (handleSubmit).
  • Action Buttons: Provides buttons for actions such as connecting with a team (AdminConnectWithTeam), deleting the user account, and toggling between edit and save mode.

    • Functionality

  • Fetching User Details: Uses useEffect to fetch user details based on type and userId from the API when the component mounts.
  • Editing User Details: Handles form submission (handleSubmit) to update user information via a PUT request to the API.
  • Deleting User Account: Implements deleteUser function to delete the user account through a DELETE request to the API.
  • Navigation: Provides a "Back" button to navigate back to the user list page (/admin/users/${type}) using Link from Next.js.
