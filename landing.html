Features of Landing Page

    The provided code includes two React components: Index and UserLayout, which are used to structure and display various sections of a webpage. Here's an overview of each component:

    Index Component

    The Index component serves as the main page structure, incorporating various custom components to build the webpage's content. It includes:
  • Hero: A prominent section usually featuring a large banner or introductory content.
  • SectionWrapper: A wrapper component that applies consistent styling (using the cn function for dynamic class names) to the contained sections.
  • AboutUs: A section providing information about the company or service.
  • Categories: A section likely displaying different categories of products or services.
  • Inspiration: A section designed to inspire visitors, possibly with ideas, tips, or showcases.
  • Working: A section detailing the working process or methodology.
  • ClientReview: A section showcasing client reviews and testimonials.
  • Newsletter: A section encouraging visitors to subscribe to a newsletter.
  • ChatBot: A section integrating a chatbot for visitor interaction and support.

  • NextTopLoader: A top-loading bar to indicate page loading progress, customized with a specific color and without a spinner.
  • Navbar: A navigation bar for site-wide navigation.
  • Footer: A footer for the page, likely containing additional navigation links, contact information, and other relevant details.
