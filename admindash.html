AdminLayout Component Documentation

    The AdminLayout component is a React functional component that provides a structured layout for admin-related pages. It includes a top loader, a navbar, a sidebar, and a main content area for children components.

    Dependencies:

    The component imports:
  • AdminSidebar from the custom admin components directory.
  • Navbar from the custom components directory.
  • NextTopLoader from nextjs-toploader for displaying a loading bar at the top of the page.

    • Props

  • children: ReactNode - The content to be rendered within the main section of the layout.

    • Component Structure

    The AdminLayout component returns a JSX structure consisting of:
  • NextTopLoader: A top loader component that displays a loading bar at the top of the page.
    showSpinner={false}: Disables the spinner.
    color='#2563eb': Sets the color of the loading bar to a shade of blue.
  • main: The main container element with a class of admin.
    Uses a CSS grid to structure the layout.
    The grid has two columns and two rows, with gap adjustments for medium screens and above (md:gap-8).
    Contains a header for the navbar, a sidebar, and a section for the main content.

    • Layout Description

  • Navbar: Placed in the header section and spans across all columns. Has additional styling with max-w-screen-2xl and shadow-md.
  • AdminSidebar: Positioned in the first column and has a margin adjustment for medium screens (md:ml-8).
  • Main Content Section: Occupies the second column and provides a container for the children components. Styled with bg-white, shadow-md, p-4 for padding, and md:p-8 for additional padding on medium screens and above. Includes overflow-y-auto to allow scrolling of content if it exceeds the viewport height. Has a margin adjustment on the right for medium screens (md:mr-8).
